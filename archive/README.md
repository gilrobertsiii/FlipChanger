# Archive Directory

This directory contains archived development artifacts and test applications that are no longer actively used.

## Contents

### `hello-world-apps/`
- **hello-world-app/**: JavaScript/TypeScript Hello World test app (initial learning/testing)
- **hello-world-ufbt/**: C/C++ Hello World test app (uFBT setup verification)

These apps were used to verify the development environment and learn Flipper Zero app development basics. They are preserved for reference but are not part of the main FlipChanger application.

## Main Application

The main FlipChanger application is located in `/flipchanger-app/`
